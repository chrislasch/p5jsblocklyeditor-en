Tutorial: Change Block View

Start Example Program:
Fit Blocks to Available Editor Area

  • In the editor, there is a toolbar at the bottom right. If you click on the top symbol, all existing blocks will be fitted to the available editor area.

    If there are only a few blocks, they will be displayed greatly enlarged.

    If there are many blocks, they will be displayed smaller so that all are visible.

Reset View

  • If you click on the second symbol from the top, all existing blocks will be displayed in normal size.

Zoom In and Out

  • To zoom in or out, press the "Ctrl" key and simultaneously rotate the middle scroll wheel of the mouse.

    With the third and fourth symbols from the top on the toolbar, you can gradually zoom out or zoom in.

Automatically Arrange Blocks

  • If many blocks are randomly placed in the editor, you can have them automatically arranged vertically.

    To do this, click with the right mouse button on an empty editor area and then click with the left mouse button on "Clean up blocks".

    The blocks will then be automatically arranged.

Collapse and Expand Blocks

  • A block can be collapsed. To do this, click with the right mouse button on the block and then click with the left mouse button on "Collapse block".

    The collapsed block only takes up one line. This way you can also accommodate very extensive programs in the editor and only edit individual blocks.

    A block can be expanded again. To do this, click with the right mouse button on the block and then click with the left mouse button on "Expand block".
