Tutorial: Save and Load

Start Example Program:
Save a Program

  • To save a program, replace "BlocklyCode" in the input field at the bottom left with a filename of your choice.

    After entering a filename, click on the "Save" button. Caution: If you press the "Enter" key, the webpage will reload, so please click "Save" to save.

    The extension of a p5.js-Blockly program is: ".p5xml" and is automatically appended to your filename.

Load a Program

  • To load a program, click on the "Load Program" button in the middle at the bottom.

  • A dialog opens. Click on the program file you want to load. Only files with the extension ".p5xml" can be loaded.

    After clicking on the desired program file, click on the "Open" button and the program will be loaded.

    Once the program is loaded, click on the "Execute Code" button to start the loaded program.
