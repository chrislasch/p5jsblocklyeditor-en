Tutorial: Generate JavaScript

Start Example Program:
Display JavaScript

  • Block programming is the preliminary stage to programming with program text. The programming language of the p5.js-Blockly-Editor is JavaScript.

    To generate JavaScript from the blocks, click on the "JavaScript Code" button in the middle at the top.

    A window opens in which the program is displayed in the programming language "JavaScript".

Execute the Program in Text Mode

  • The program programmed with blocks can be opened, edited, and executed with a text editor. To do this, click on the "Text Editor" button in the middle at the top.

    A new window opens and the JavaScript code generated from the blocks is displayed and can be executed.
