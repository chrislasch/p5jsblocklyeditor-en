K1.5a: Variable Properties

Define a Variable

  • 4 circles should be drawn. To do this, you build the following program:

  • Now the diameter of the 4 circles should be changed. To do this, 8 numbers would have to be changed. That's tedious.

    It would be easier if you put a placeholder (variable) in the fields for "Width" and "Height" and assign a value to this variable. If you later want to change the diameter, you only need to change one place in the program.

    To define this variable, you click on the "Variables" category and there on "Create variable ...". In the dialog, you enter the name "diameter" and should then see the following:

  • The block "set diameter to" is placed at the beginning of the "setup()" block and the number 100 is attached and the "diameter" block is built into the empty fields:

  • Now you can easily change the diameter of the circles by changing the number at the "set diameter to ..." block.
