  • Such repetitions can be combined with a repetition block. Since the x-coordinate of the 4 circles changes, we need a new variable for the x-coordinate. The value of the variable "xcoord" is changed on each pass through the repetition loop.

 K1.6b: Repetition of a Repetition

Multiple Rows of Circles

  • If you build a repetition block into a repetition block, then multiple rows of circles can be drawn. For this we need another variable for the y-coordinate.

  • The outer repetition block is repeated 4 times. The inner repetition block is repeated 4 x 4 = 16 times. So that the circles are always drawn from the beginning of the line, the x-coordinate must be reset to 50 before each restart of the inner repetition block.
