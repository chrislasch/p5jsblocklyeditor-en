K1.12: Import Images

Convert Image:

  • For an image to be inserted in the p5.js Blockly Editor, it must be encoded in Base64 format. There are online tools for this. For example: https://www.base64-image.de/

  • Upload an image in JPG, PNG, GIF, WebP, SVG or BMP format to the encoding tool and copy the generated code to the clipboard.

    In the p5.js Blockly Editor, insert the generated code into the Base64 text field of the example program and run the program. You can start the example program below:
