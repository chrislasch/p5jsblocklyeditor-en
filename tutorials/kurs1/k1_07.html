K1.7a: Gaps in the Trail

Frames and the Mouse

  • With the mouse, you can paint something on the drawing area by entering the x and y coordinates of the mouse as coordinates for a circle. The computer determines once per frame where the mouse is currently located and paints a circle there.

    If you move the mouse slowly, you can paint a continuous line. If you move the mouse quickly, gaps appear in the line because the mouse coordinates change too quickly.

 K1.7b: Line Without Gaps

New and Old Mouse Position

  • A continuous line can be drawn if you draw a line instead of a circle and use it to connect the old mouse position in the last frame and the new mouse position in the current frame.

  • The mouse coordinates in the last frame are stored in the blocks "Mouse: previous x-coordinate" and "Mouse: previous y-coordinate".

 K1.7c: Variable Line Thickness

Line Thickness Depending on Mouse Speed

  • Creative programming is fun when you build interesting effects into your programs. In the following example, the line thickness is changed depending on the mouse speed.

  • This is achieved by calculating the line thickness as the distance between the old and new mouse position.

    If you move the mouse slowly, this distance is small and the line thickness is also small. If you move the mouse quickly, this distance is large and the line becomes thicker.
