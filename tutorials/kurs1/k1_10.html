K1.10a: Display Text

Left-aligned, Right-aligned, Centered

  • Text is displayed at the specified position. The text size can be set and you can display the text left-aligned, right-aligned, and centered. The appearance of the text can be determined with line thickness, line color, and fill color.

 K1.10h: Input Text

Input Text and Store in a Variable

  • You can insert one or more text input fields into the drawing area. There you can type text that is stored in a variable. The stored text can be output again.
