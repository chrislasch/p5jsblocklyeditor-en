S1.1: Turtle's First Steps

The Basic Program

  • For the turtle to be used in a program, it must be inserted in a specific way. A minimal turtle program looks like this:

    The turtle is placed on the canvas at the desired position (x/y) with "Insert Turtle".

    This is followed by instructions that control the turtle's movement.

    With the "Turtle: Record Movement" block, the turtle's movement is saved.

    In the "draw()" block, insert the "Background Color" block at the beginning so that the canvas is cleared on each pass.

    With the "Turtle: Start Animation" block, the recorded movement of the turtle is executed animated. The number at "Speed" determines how fast the turtle moves.
