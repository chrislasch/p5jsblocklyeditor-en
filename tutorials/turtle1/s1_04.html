Change Side Length of Square

  • If the distance the turtle moves forward is shortened or lengthened, differently sized squares are created.

    For each square, you set the side length and create a repeat loop to draw the square.

Load Example Program:
Change Side Length with a Variable

  • A quantity that changes during program execution is called a variable. The side length of the square should be changed using a variable.

    To define a variable, click on "Variables" in the toolbar. The variables needed for the canvas and turtle are already present. These should not be changed so that the program can run without errors.

    To define a new variable, click on "Create variable..." and enter a variable name such as "sideLength". No umlauts or special characters should be used in the variable name.

    The new variable "sideLength" is added to the list of existing variables.

    The variable "sideLength" is inserted into the program by inserting the "set xxxxxx to" block into the program. The "xxxxxx" stands for any variable name.

    If "sideLength" is not already displayed as the variable name, click on the variable name and click on the variable "sideLength" in the list of variables.

    The value of the new variable should be set to the number 10. To do this, attach the number block from the "Mathematics" category to the variable and set the number value to 10.

    Finally, replace the number value in the "Forward" block with the variable name by getting the "sideLength" block from the "Variables" category and replacing the number value in the "Forward" block with it.

    We no longer need the other two repeat blocks. These can be deleted. To do this, drag the "Record Movement" block under the first repeat block and move the other two repeat blocks to the trash.

    The program should now look like this. The side length can be changed by changing the number value in the "set sideLength to" block.
