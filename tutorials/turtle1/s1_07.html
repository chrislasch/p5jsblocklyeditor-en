Outsource Instructions to Function

  • In the template, the turtle draws two squares:

    In more complex programs, the instructions for the turtle can become more extensive. Instructions can be outsourced to a "function". In the program flow, you only insert the name of the function and all instructions that are in the function are executed.

    A function is defined by dragging the "do something" block from the "js-Functions" category into the editor.

    Replace the expression "do something" with a designation that describes what the function should accomplish. Here the function should draw a square, so the function gets the name "drawSquare".

    All instructions with which a square is drawn are inserted into the function "drawSquare".

    The name of the function is then inserted into the program at the places where a square should be drawn. The function block with the name "drawSquare" can be found in the "js-Functions" category once you have defined the function.
