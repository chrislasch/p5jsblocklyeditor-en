A Spiral

  • The turtle should draw a spiral. A spiral looks like this, for example:

    When you look at the spiral, you notice that the distance the turtle moves forward becomes longer after each direction change. The direction angle is 45°. This should now be taught to the turtle.

    In the program, a variable "lineLength" is defined, which is increased after each direction change. To do this, click on "Create variable..." in the "Variables" category and name the new variable e.g. "lineLength".

    The "set lineLength to" block is inserted into the program and supplemented with the value 0

    The repeat loop is repeated 150 times and the rotation angle is set to 45°.
