S1.8 - Recursion

Load Template:
Draw a Tree

  • The turtle should draw a tree. To do this, a function "drawTree" is defined. A value can be passed to a function with which the function then calculates.

    A parameter is defined by clicking on the gear symbol in the function block and dragging the left "Variable" block with the mouse to the right "Parameter" block.

    The variable can be given its own name. The turtle should draw a tree where the branch length changes, so the variable gets the name "branchLength".

    In the function list, the function "drawTree" appears as the following block:

    A value can be docked to the docking point. The starting length of a branch should be 75 pixels long, so the value 75 is docked. The number block can be found in the "Mathematics" category.

    In the function "drawTree", the turtle is taught how to draw the tree. First, the pen is put down so that the turtle draws.

    The branches of the tree should always become shorter the higher the tree is. But at a certain length of a branch, the turtle should not draw any further.

    With the help of an "if-do" query, the length of the branch is compared with a limit and the turtle only draws the tree if the branch length is greater than the set limit.

    A tree has very many branches. If these all had to be programmed individually, then the necessary program would be very very long. With a brilliant programming trick, you can program a tree with very many branches with only a few blocks.

    The function "drawTree" calls itself again. This is called recursion. When called again, a smaller value is passed for the branch length, so that the branches always become shorter.

    So the turtle draws a branch and turns 20° to the right until the branch length has become smaller than 5 pixels. In doing so, the function "drawTree" is called 7 times, because there are 7 branches that are placed one after the other at an angle of 20° and always become shorter.

    As soon as the end of the tree is reached, the program jumps from the 7th level of the function "drawTree" back to the 6th level and executes the further instructions.

    When everything is done at the 6th level, the program jumps back to the 5th level and continues working there. This repeats until the 1st level of the function "drawTree" has been processed and the turtle is back at the root of the tree.
