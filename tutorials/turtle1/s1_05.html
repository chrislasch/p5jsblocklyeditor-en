Draw Regular Polygons

  • In a triangle, the sum of the interior angles is 180°.

    To build a quadrilateral from a triangle, a fourth corner is added. A quadrilateral consists of two triangles, which is why the sum of the interior angles in a quadrilateral is 360°.

    In a pentagon, another triangle is added to the quadrilateral, which is why the sum of interior angles in a pentagon is 360° + 180° = 540°.

    In a hexagon, another triangle is added to the pentagon, which is why the sum of interior angles in a hexagon is 540° + 180° = 720°.

    In general, one can say that an N-gon has a sum of interior angles of (N-2) · 180°.

    If the turtle is to draw an N-gon, then it can calculate for itself how large the angle must be by which it should change its direction. But we must remember that we subtract the desired interior angle from 180° to get the turtle's direction angle:

    In addition to the variable "sideLength", the variables "numberOfSides", "sumOfInteriorAngles" and "directionAngle" are inserted:

    For the sum of interior angles: "sumOfInteriorAngles" = ("numberOfSides"-2)·180°

    In Blockly there are no parentheses. When calculating a term, the rule applies that two numbers that are in a common block are calculated first. In the following expression, "numberOfSides"-2 is calculated first and then the result is multiplied by 180. The calculation block can be found in the "Mathematics" category.

    For an interior angle in a regular polygon: interior angle = sumOfInteriorAngles / numberOfSides. To calculate the turtle's direction angle, we must subtract the interior angle from 180°:

    "directionAngle" = 180° - ("sumOfInteriorAngles" / "numberOfSides")

    The calculated direction angle is entered in the "Turn Left" block.

    For the turtle to draw an N-gon, the number of sides is entered as the number of repetitions.

    The complete program then looks like this. If you change the number of sides, the turtle draws a corresponding polygon.
