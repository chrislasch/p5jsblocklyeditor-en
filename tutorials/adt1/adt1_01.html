ADT1.01: From Variables to Lists/Arrays

Variables

  • The four color names for the following circles should be stored in variables.

    To do this, you click on "Create variable..." in the "Variables" category and create four variables with the names "color1" to "color4".

    To assign a color name to a variable, you drag the "set colorX to" block into "setup()".

    The variables are assigned the desired color names by docking the text block from the "Text" category and typing the desired color name.

    To set the fill color or line color, a variable block is dragged from the "Variables" category,

A List of Values

  • The color names for the four circles can also be stored in a single variable.

    To do this, you create a variable with, for example, the name "colorList".

    To fill the list with values, you drag the "set colorList to" block into "setup()" and dock the "create list with" block from the "js-List" category.

    To add more elements to the list, you click on the gear symbol at the top left and drag an element block into the list.

    To set the fill color of a circle, you dock the "in list list take the ... element" block from the "List" category to the "Fill Color" block and insert the desired number.
