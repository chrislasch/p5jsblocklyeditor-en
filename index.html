About p5.js-Blockly-Editor

The p5.js-Blockly-Editor enables graphics programming based on the p5.js library using blocks. JavaScript code can be generated from the blocks, which can then be executed, for example, in the p5.js-Flems-Editor or in the p5.js Web Editor .

The p5.js-Blockly-Editor is a project accompanying computer science lessons at Gymnasium Westerstede .
